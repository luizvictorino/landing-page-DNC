## Instalação

1. Clone o repositório para o seu ambiente local:

   ```bash
   git clone https://github.com/luizvictorino/landing-page-DNC.git
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd landing-page-DNC
   ```

3. Abra o arquivo `index.html` no seu navegador preferido.
